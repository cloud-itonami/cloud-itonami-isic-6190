# Operator Guide

## First Deployment

1. Register number ranges, SIP endpoints and responsible operators.
2. Import existing number assignments and CDR history.
3. Run read-only E.164 and SIP endpoint validation.
4. Configure routing, billing and escalation paths.
5. Publish a dry-run billing run and audit export.

## Minimum Production Controls

- E.164 validation before any number activation or routing
- SIP endpoint validation before call dispatch
- CDR and SMS record integrity for every billable event
- audit export for every provision and disclosure
- backup manual routing and notification process

## Certification

Certified operators must prove numbering integrity, evidence-backed
billing, lawful-intercept readiness and human review for
provisioning-affecting actions.
